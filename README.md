# Sorting Visualizer

Welcome to Sorting Visualizer, a web application built using React.js that allows you to visually experience different sorting algorithms. 

![Sorting Visualizer Demo](public/demo.gif)

## Tech Stack
- React.js

## Features

### Sorting Algorithms
1. **Bubble Sort**
2. **Insertion Sort**
3. **Merge Sort**
4. **Quick Sort**
